### Common Arguments and Options
- **`-t`**: Key type
- **`-b`**: Number of bits
- **`-C`**: Comment/label
- **`-f`**: Output file
- **`-N`**: Passphrase
- **`-q`**: Quiet mode
- **`-y`**: Extract public key
